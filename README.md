# ChoreKey for Windows & Mac

Kids' apps and games stay locked until their chores are done and approved by a parent. Works fully offline. No account, no analytics, no cloud.

## Download

Get the latest installers on the [**Releases**](../../releases/latest) page:

- **Windows (Intel/AMD, most PCs):** `ChoreKeySetup-x64.exe`
- **Windows (ARM):** `ChoreKeySetup-arm64.exe`
- **macOS (Apple Silicon):** `ChoreKey-macos-arm64.zip`

All builds are self-contained, so you don't need to install anything else.

### Install
- **Windows:** run the installer (per-user, no admin needed). ChoreKey appears in your Start Menu.
- **macOS:** unzip and drag **ChoreKey.app** to Applications. It's signed and notarized by KyrosWorks LLC, so it opens with no security warning.

## Privacy
Everything stays on your computer. No accounts, no tracking, no data sales. Phone, Messages, and Maps equivalents are never blocked.
